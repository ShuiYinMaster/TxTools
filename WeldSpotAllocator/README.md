# 焊点分配 / 更新（WeldSpotAllocator）使用帮助

TxTools 套件中的 Process Simulate 2402 插件，用于在焊接操作之间**更新焊点位置**或**分配焊点**。适用三类场景：旧版轨迹换新版坐标、把待分配焊点塞进已调好的参考轨迹、左右对称分配。

---

## 一、三种模式

打开窗口后顶部「模式（单选）」三选一，下方灰字会提示该模式的两个集合含义。

### A 位置更新
- **参考集**＝旧版焊接操作，**目标集**＝新版焊接操作。
- 作用：把旧版焊点的位置更新到新版坐标，**分配关系不变**（不新建轨迹）。
- 适用：同一批焊点出了新版坐标，要把现有轨迹里的焊点挪到新位置。

### B 新焊点分配
- **参考集**＝已分配好的参考操作，**目标集**＝待分配焊点清单（也是焊接操作）。
- 作用：为每条参考轨迹 Paste 出一条 `_Mapped` 新轨迹，把待分配焊点**移入**新轨迹，继承参考的姿态 / 过渡点 / 轨迹参数。
- 适用：新车型焊点要复用旧轨迹已调好的机器人姿态和参数。

### C 对称分配
- **参考集**＝左侧参考操作，**目标集**＝右侧待分配焊点清单（操作）。
- 作用：参考轨迹关于车身中线镜像后与右侧待分配焊点匹配，过渡点（via）镜像复制到 `_Mapped`。
- 适用：左右对称的焊点，用左侧已调好的轨迹生成右侧。

---

## 二、操作选择（参考集 / 目标集）

两个网格分别装参考集与目标集，**只接受焊接操作**，非焊接操作会被自动剔除。

- **加入对象**：点网格内的拾取行（绿色行），再去 PS 的对象树或 3D 视图里点选操作即可加入；计数会实时更新。
- **交换参考/目标**：一键对调两个网格内容（A 模式调换旧↔新方向，B/C 调换参考与待分配清单）。
- **清空列表**：清空两个网格 + 预览表。

---

## 三、匹配条件 + 分配后处理

### 匹配条件
- **焊点距离匹配** + **距离阈值 mm**：按空间距离匹配，超过阈值不配对。默认勾选，阈值 5 mm。
- **焊点名匹配** + **剥前缀(名)**：按焊点名匹配；可填要剥掉的名称前缀再比对。
- 两个开关可单勾或同勾（同勾＝距离与名都满足才算匹配）；都不勾时按仅距离处理。
- **不同车件参考系**（B/C）：参考在工位、新点在导入原点对不上时勾选，会把参考点转到车件局部系再匹配。**要求目标车身在世界原点。**

### 分配后处理
- **复制旋转姿态**（B/C）：勾选则把参考焊点的姿态赋给移入的焊点（保持焊点自身位置）；不勾保留焊点自身法向。
- **镜像翻轴(X前后/Y左右/Z上下)**（C）：镜像后焊钳绕工具自身某轴翻 180° 的修正项。焊钳坐标约定为 X＝前后、Y＝左右、Z＝上下。默认 None，需按实际枪型实测选择。
- **镜像过渡点(C)**：把参考轨迹的过渡点镜像复制到 `_Mapped`。
- **复制轨迹参数**（B/C）：把参考焊点的轨迹参数复制到移入的焊点。
- **挪动焊点(移入新轨迹,不复制)**（B/C）：把待分配焊点本身移入新轨迹，而非复制——焊点 MFG 全程被引用，不会变灰。
- **操作数量严格(A)**：操作配对时要求两侧焊点数量严格相等。
- **仅更新位置(A)**：只更新焊点坐标，不改其它属性。

---

## 四、动作按钮

- **预览匹配**：执行匹配并把结果填入下方预览表，不写入场景。
- **执行写入**：把预览结果写入 PS（A 改焊点坐标；B/C Paste 出 `_Mapped` 并移入焊点）。需先预览；执行前有确认弹窗，可用 PS 撤销回退。
- **导出Excel**：把预览表内容导出到 Excel。
- **交换参考/目标**、**清空列表**：见上文。

**预览结果表**列含义：新轨迹 / 参考焊点 / 待分配焊点 / 依据（距离或名，镜像时标「·变换」）/ 距离mm。

---

## 五、匹配逻辑（多条轨迹时）

### A 模式：操作一对一配对
每条参考操作从目标操作里挑一个搭档（挑中的不再被别的参考操作选）。配对优先级：焊点数量 → 操作名相等 → 质心距离最近。配对后操作内部焊点取最近的未用焊点。参考轨迹之间互不干扰。

### B / C 模式：目标焊点共享全局池
所有目标操作的焊点展平成一个池。按参考集顺序遍历每条参考轨迹，对其每个焊点（C / 不同车件参考系会先镜像或转局部系）在整个池里取最近的未用焊点。**一个待分配焊点被取走后不再参与其它匹配**，全局一对一、不会重复分配。

> 注意：B/C 是按参考集顺序的贪心匹配——若两条参考轨迹的焊点都靠近同一个待分配点，先处理的那条拿走。点分布密集或接近阈值边界时可能不是整体最优。

---

## 六、注意事项

- 两个网格**只接受焊接操作**，拾取到其它对象会被自动剔除并记日志。
- B/C 是**移动**待分配焊点（不是复制后删除），焊点 MFG 不会因失去引用而变灰。
- 「不同车件参考系」要求**目标车身在世界原点**。
- 镜像翻轴需按实际焊钳实测确定（None / X / Y / Z）。
- 执行写入可用 **PS 撤销**回退。
- 首次运行可关注日志中的 `[Grid]`（网格就绪）、`[Move]`（移动焊点用的 API）、`[Writer]`（写入统计）等提示。

---

## 七、典型流程

**B 新焊点分配为例：**
1. 选「B 新焊点分配」。
2. 参考集网格拾取已分配的参考操作；目标集网格拾取待分配焊点清单操作。
3. 设匹配条件（距离阈值；对不上原点时勾「不同车件参考系」并确认目标车身在世界原点）。
4. 按需勾「复制旋转姿态 / 复制轨迹参数 / 挪动焊点」。
5. 点「预览匹配」，核对预览表的配对与距离。
6. 确认无误点「执行写入」，检查生成的 `_Mapped` 轨迹；如需可「导出Excel」留档。
